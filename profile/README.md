## About

DiCRA is a digital public good providing open access to geospatial datasets and analytics that strengthen evidence-driven policy making for climate resilient food systems.

The platform combines remote sensing, machine learning, and citizen-science validation to identify climate-resilient and climate-vulnerable farms across millions of hectares. It is curated by hundreds of data scientists and citizen scientists worldwide.

Built on the principles of open access, open software, open code, and open APIs.

<br/>

## Platform

DiCRA 2.0 is a federation of thematic sub-portals built on shared core services. Each sub-portal focuses on a specific agricultural domain — its own data, analytics, and interface — drawing from common infrastructure for cataloging, storage, and APIs.

<br/>

### Applications

End-user interfaces for exploring DiCRA, browsing the catalog, and interacting with platform tools.

- [**dicra-website**](../../dicra-website) — Landing site for dicra.org
- [**dicra-geospatial-frontend**](../../dicra-geospatial-frontend) — Main platform interface
- [**dicra-stac-browser**](../../dicra-stac-browser) — STAC catalog browser
- [**dicra-plp**](../../dicra-plp) — AI assistant for NABARD's Potential Linked Plans
- [**dicra-documentation**](../../dicra-documentation) — Platform documentation

<br/>

### Core services

Backend services and pipelines that power the platform.

- [**dicra-geospatial-backend**](../../dicra-geospatial-backend) — Platform API server
- [**dicra-stac-server**](../../dicra-stac-server) — STAC-compliant catalog API
- [**dicra-etl**](../../dicra-etl) — Ingestion and transformation pipelines

<br/>

### Sub-portals

Domain-focused sub-portals combining their own interface, datasets, and analytics.

- [**dicra-climate**](../../dicra-climate) — Climate datasets and resilience models
- [**dicra-weather**](../../dicra-weather) — Weather integration and forecasting
- [**dicra-livestock**](../../dicra-livestock) — Livestock datasets and analytics
- [**dicra-crop-yield-estimation**](../../dicra-crop-yield-estimation) — Crop yield models and tooling
